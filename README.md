# Dobrodošli na BMDb 🎥

**BMDb (Bosnian Movie Database)** je centralna platforma namijenjena arhiviranju, ocjenjivanju i praćenju svih filmskih i televizijskih ostvarenja iz cijelog svijeta. Naš cilj je pružiti najpreciznije informacije o domaćim i svjetskom produkcijama na bosanskom jezikuss.
